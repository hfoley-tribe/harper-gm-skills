## Format rules

- **Phone-readable.** Whole thing should fit a phone screen with one scroll. If it doesn't, cut the watching section before cutting key-deals.
- **Color tags per engagement in the GTM block:** 🟦 / 🟨 / 🟩 (3 active). 🟪 reserved for workstreams / cross-cutting. Order accounts by time-urgency for the day, not priority-stack order.
- **Priorities block is a personal note, not a team-facing post.** No SOP today says these go to a Slack channel. Block content is for Harper's self-direction Monday morning; if Brandon or a sales-team SOP ever names a channel, hard-code it in `SKILL.md` and add a "paste-ready" constraint then.
- **GTM Weekly block is talking-point shape.** Bold lead + one sentence of context. Harper speaks from this; don't write paragraphs.
- **Cite every item.** Source citation in parens. No source-less bullets.
- **Voice:** same as Sunday DM. Direct, terse, operator-grade. No filler.
- **No em dashes anywhere** (per Harper's global preferences). Hyphens or rewrites only.
